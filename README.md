## Name
Taiwo Bolaji

## Project Title
Smartphone Cybersecurity Audit

## Objective
The aim of this project is to identify basic cybersecurity weaknesses in a smartphone and suggest practical security improvements.

## Scope
I audited my smartphone.

## Tools Used
- Phone settings
- Security settings
- System update page
- App permissions page

## Files in this Repository
- checklist.md
- findings-report.md
- recommendations.md
- screenshots/

## Key Findings
- Phone lock is enabled
- Some updates are missing
- App permissions are too open
- Two-factor authentication is not fully enabled

## Conclusion
This project helped me understand how to identify common cybersecurity risks and apply basic security improvements.
